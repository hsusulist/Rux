Features

Everything you need to build faster

Rux gives your AI full access to your Roblox Studio session through a secure plugin bridge.

Script read & write
The AI can read, create, edit, and delete any script in your game — with automatic snapshots before every change.
Agent mode
Give the AI a multi-step task. It drafts a plan for your review, then executes it tool-by-tool inside Studio.
Code search
Search across every script for variables, functions, or patterns. The AI finds usages and explains context.
Multi-model AI
Switch between Gemini 3 Fast, Gemini 3 Pro, Claude Sonnet 4.6, and Claude Opus 4.6 per conversation.
Saved conversations
Conversations are saved locally in your browser. Resume any past session right from the sidebar.
Snapshot & restore
Every write operation creates a snapshot. Restore any script to its previous state with a single command.

How it works

Up and running in minutes

Install the Studio plugin, connect it to your Rux URL, and start chatting.

1
Install the Roblox Studio plugin
Add the Rux plugin script to your Roblox Studio. It runs silently in the background and establishes a secure connection to the server.
2
Set your server URL in the plugin
Open the plugin script and set SERVER_BASE_URL to this app's URL.
local SERVER_BASE_URL = "https://your-rux.replit.app"
3
Add your API keys
Set ANTHROPIC_API_KEY (and optionally GOOGLE_API_KEY for Gemini models) in your Replit secrets.
4
Start chatting
Open Rux in your browser. The plugin status turns green and your entire Roblox Studio session is available to the AI.
