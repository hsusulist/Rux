Features

Everything you need to build faster

Rux gives your AI full access to your Roblox Studio session through a secure plugin bridge.

Script read and write
Read, create, edit, and delete any script in your game with automatic snapshots before every change.
Agent mode
Give the AI a multi-step task. It drafts a plan for your review, then executes it tool-by-tool inside Studio.
Code search
Search across every script for variables, functions, or patterns. Find usages and explain context.
Multi-model AI
Switch between Gemini, Claude, GPT, Qwen3, and more per conversation. Free models available.
Saved conversations
Conversations are saved automatically. Resume any past session from the sidebar at any time.
Checkpoint and restore
Every agent run creates a checkpoint automatically. Restore any checkpoint from the web UI.
Instance explorer
Browse the game hierarchy, get and set properties, find instances by name.
Error diagnostics
Check scripts for syntax issues, view output and error logs directly from Studio.

How it works

Up and running in minutes

Install the plugin, enter a code, and start chatting. No API keys needed.

1
Install the Studio plugin
Add the Rux plugin from the Roblox Creator Store. It runs in the background and connects to the server.
2
Get your connection code
Click "Connect Studio" in the web sidebar and copy the 4-character code.
Example: A7K3
3
No API keys needed
Rux handles all AI integrations server-side. Claude, Gemini, GPT, and more work out of the box.
4
Start chatting
Your entire Studio session is available to the AI. Choose Chat for quick questions or Agent for multi-step tasks.
Security

Your code stays safe

Rux is designed with security at every layer.

Encrypted passwords
All passwords hashed with bcrypt. No plaintext stored or transmitted.
Server-side AI keys
API keys never reach your browser or Studio. All calls are proxied server-side.
Auto-checkpoints
Every agent task saves script state before modifications. Restore any time.
