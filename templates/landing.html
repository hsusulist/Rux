Features

Everything you need to build faster

Rux gives your AI full access to your Roblox Studio session through a secure plugin bridge.

Script read and write
Read, create, edit, and delete any script in your game with automatic snapshots before every change.
Agent mode
Give the AI a multi-step task. It drafts a plan for your review, then executes it tool-by-tool inside Studio.
Code search
Search across every script for variables, functions, or patterns. Find usages and explain context.
Multi-model AI
Switch between Gemini Flash, Gemini Pro, Claude Sonnet, and Claude Opus per conversation.
Saved conversations
Conversations are saved locally in your browser. Resume any past session from the sidebar.
Snapshot and restore
Every write operation creates a snapshot. Restore any script to its previous state with a single command.
How it works

Up and running in minutes

Install the Studio plugin, connect it to your Rux URL, and start chatting.

1
Install the Roblox Studio plugin
Add the Rux plugin script to your Roblox Studio. It runs in the background and establishes a connection to the server.
2
Set your server URL in the plugin
Open the plugin script and set SERVER_BASE_URL to your app's URL.
local SERVER_BASE_URL = "https://your-rux.app"
3
AI is ready — no keys needed
Rux uses Replit's built-in AI integrations. Claude and Gemini work out of the box, billed to your Replit credits.
4
Start chatting
Open Rux in your browser. The plugin status indicator turns on and your entire Studio session is available to the AI.
